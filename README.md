# ⚜️ Sacred KG — Backend API

FastAPI backend for Sacred KG — sacred places and tourist routes of Kyrgyzstan.

- **API:** `/api/places`, `/api/auth`, `/api/ai/chat`, `/api/bookings`, `/api/reviews`, `/api/posts`
- **Admin:** `/admin/places`, `/admin/users`, `/admin/bookings`, `/admin/ai-config`
- **AI:** DeepSeek-powered AI guide (Atashka/Apashka modes)
- **Database:** Supabase (PostgreSQL)
- **Auth:** JWT tokens + admin token
